# BookmarkSaver

A simple Bookmark Manager built with **HTML, CSS and JavaScript**.

This project allows you to:
- Add bookmarks (name + URL)
- Automatically save them in `localStorage`
- Load bookmarks on page refresh
- Edit existing bookmarks via modal
- Delete bookmarks
- Open bookmarks in a new tab

---

## 🚀 Features

### ➕ Add Bookmark
You can add a bookmark by entering:
- Name
- URL (http/https is automatically fixed if missing)

### 💾 Persistent Storage
All bookmarks are stored in the browser using `localStorage`, so they remain after refresh.

### ✏️ Edit Bookmarks
You can edit existing bookmarks using a modal window.

### ❌ Remove Bookmarks
Each bookmark can be deleted individually.

### 🔗 Quick Access
Clicking a bookmark opens it in a new tab.

---

## 🧠 Tech Stack
- HTML5
- CSS3 (modern dark UI design)
- Vanilla JavaScript (DOM + localStorage)
